# Suggested Commands
Below are the most important commands to operate within this Darwin (macOS) environment for this Kotlin project:

## Build & Run
- **Build the project**: `./gradlew build`
- **Run the Web App**: `./gradlew composeApp:wasmJsBrowserDevelopmentRun` (Standard gradle task for Wasm JS Compose apps)
- **Clean build directory**: `./gradlew clean`

## System/Utility (Darwin)
Since this project runs on Darwin (macOS), standard Unix utilities apply:
- **Search for text**: `grep -rn "pattern" .`
- **Find files**: `find . -name "*.kt"`
- **Version Control**: `git status`, `git add`, `git commit`
